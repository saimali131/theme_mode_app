# 📱 Project Overview

The Flutter Theme Mode App is a lightweight application built to demonstrate Flutter's built-in theme management capabilities. Users can switch between Light Mode and Dark Mode, with the interface updating instantly to reflect the selected theme.

This project was developed to understand Flutter's `ThemeData`, `ThemeMode`, widget rebuilding, and state management fundamentals.

# 🚀 Getting Started

Clone the repository

```bash
git clone https://github.com/saimali131/theme_mode_app.git
```

Navigate to the project

```bash
cd theme_mode_app
```

Install dependencies

```bash
flutter pub get
```

Run the application

```bash
flutter run
```
